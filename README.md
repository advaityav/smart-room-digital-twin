This project presents the design and implementation of a real-time digital twin of an intelligent room environment. An ESP32 microcontroller collects environmental data from sensors such as an LDR and transmits it to a Unity-based 3D simulation via serial communication. The virtual model dynamically mirrors real-world conditions by adjusting lighting intensity and fan speed in real time. The system demonstrates the integration of embedded systems, IoT, and interactive visualization to create a responsive and adaptive environment.
